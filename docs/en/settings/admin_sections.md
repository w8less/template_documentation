
# Admin sections

## Format implemented in `settings.json`

```json
{
    "admin": {
        "sections": {
            "hidden": [
                "legacy",
                "catalog"
            ]
        }
    }
}
```


## Description

`admin.sections.hidden` - List of admin sections tags to hide, available tags:

- legacy
- catalog
- albums
- events
- search
